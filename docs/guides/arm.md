# できること

本ページでは「ロボットアーム」機能の実装について解説します。

<br>

具体的には、以下の内容を実施します。

- ロボットアームパーツの印刷・組み立て
- サーボモータ4つによる、4軸アームの制御

!!! note サーボの割り当て
    ロボットアームはサーボモータ4つで構成され、smabo-esp32のデフォルト設定では
    - `arm_joint_1`：PCA9685 チャンネル3
    - `arm_joint_2`：PCA9685 チャンネル4
    - `arm_joint_3`：PCA9685 チャンネル5
    - `arm_joint_4`：PCA9685 チャンネル6

    が割り当てられています。軸数は、お手持ちのアームに合わせて増減できます。
