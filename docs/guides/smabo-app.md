# smabo-appとは

smabo-appはsmaboの中で

- smaboの顔
- smaboのセンサー（スマホのセンサーを使用）

の役割を担います。



## smabo-appのインストール

スマホにsmabo-appをインストールします。

なお、ここではandroidの手順のみを記載します。

!!! note
    IOSの場合、flutterのインストールなどが必要になり、少し手順が多くなるため、詳しくは[smabo-appのリポジトリ](https://github.com/smabo-smartphone-robot/smabo-app)のREADMEを確認してください。

    また、IOSの場合、アプリのインストールに、mac OSのPCが必要です。

    


スマホから[smabo-appのReleaseページ]()にアクセスします。

Releaseページの中で、一番最新のものを選択しダウンロードしてください。

ダウンロードしたapkファイルを実行するとsmabo-appがインストールされます。


# 動作手順

以下手順で、動作の確認を行います。

## smabo-brainの起動

最初に、smabo-brainを起動します。

```bash
cd ~/smabo-brain
```

```bash
python3 -m brain
```

## smabo-webの起動

次に、別ターミナルで、smabo-webを起動します。

```bash
cd ~/smabo-web
```
```bash
npm run dev
```

<br>

この状態で、ブラウザから`localhost:******`に接続し、以下のように表示されればOKです。

## smabo-appの接続

smabo-appを起動し、設定画面を開きます。

IPの欄に「smabo-brainを起動しているPC/SBCのIPアドレス」を入力し、Connectをタップします。

しばらくして「Connected」と表示されればOKです。

## smaboの顔の設定

smabo-appを起動すると、顔が表示されます。

デフォルトでは、下図のように表示されますが、「」から設定を変更できます。

設定では「目の大きさ」「目の種類」「背景色の変更」などができます。

## 各種センサーの状態を可視化

smaboの各種センサのON/OFFは左下のボタンで設定できます。

全てのセンサをONにした状態で、smabo-webにアクセスすると各種センサの情報を可視化することができます。

## 音声認識

smaboに対して「smabo」と話しかけると、smaboの目が虹色に光ります。

目が虹色の状態で何か言葉を話すと、音声認識が開始されます。

目の色が戻った後に、smabo-webの画面を確認すると先ほど話した言葉が表示されていることが確認できます。

## smaboに発話させる

smabo-webで、適当な言葉を入力し、送信ボタンをクリックするとsmaboが入力された音声を発話します。

なお、「」ボタンにて発話のON/OFF、吹き出しの有無を設定可能です。

## smaboの目を動かす

smaboの目はデフォルトでは、ランダムに動きますが、マニュアルモードだと外部からの通信で自由に動かすことができます。

smabo-webの「eye-controller」タブでコントローラを操作すると、それに合わせて目が動くことが確認できます。
