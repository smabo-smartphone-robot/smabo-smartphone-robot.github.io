# 動作計画とは

動作計画は、ロボットアームについて

- 目標とする手先の位置・姿勢から、各関節の角度を求める
- 障害物や可動範囲を考慮した、関節の動かし方を計算する

を行う機能です。

<br>

smaboでは、smabo-esp32が公開する`/joint_states`をもとに、ROS 2の動作計画フレームワーク（MoveIt2）を用いて実現します。
