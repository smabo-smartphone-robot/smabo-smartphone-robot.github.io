# ナビゲーションとは

ナビゲーションは、エンコーダによるオドメトリ（自己位置）をもとに、

- 目標地点までの経路の生成
- 経路に沿った走行制御

を行う機能です。

<br>

smaboでは、smabo-brain-rosで公開される`/odom`をもとに、ROS 2のナビゲーションスタック（Nav2）を用いて実現します。
