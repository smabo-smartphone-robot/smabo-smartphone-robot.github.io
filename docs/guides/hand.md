# できること

本ページでは「ハンド」機能の実装について解説します。

<br>

具体的には、以下の内容を実施します。

- ハンドパーツの印刷・組み立て
- サーボモータ2つによる、ハンドの開閉制御

!!! note サーボの割り当て
    ハンドはサーボモータ2つで構成され、smabo-esp32のデフォルト設定では
    - `left_hand`：PCA9685 チャンネル0
    - `right_hand`：PCA9685 チャンネル1

    が割り当てられています。

# 配線

!!! note ブレッドボードの+極について <!--重要なので、このNoteは特に目立つように-->
    SAD101購入時についていた、**+極のジャンプワイヤーは必ず外してください**。
    ついたままだと、**ESP32, アクチュエータが破損する**可能性があります。<!--太字を赤字にする-->
    ![alt text](../images/esp32_breadboard_remove_jumper.png)

<br>

最後に、「smabo-esp32のconfigファイル」にて設定したピンに従って、配線を行います。

今回の場合、以下のようになります。

※ 下記表において、「接続先1」と「接続先2」は、両者が回路的に直接繋がっていることを指します。

| 接続先1 | 接続先2 | 補足 |
| --- | --- | --- |
| config.jsonで`i2c/scl`に設定したESP32のピン番号<br>（`config.esp32-classic.json`で設定されているデフォルト値は22） | PCA9685のSCLピン | - |
| config.jsonで`i2c/sda`に設定したESP32のピン番号<br>（`config.esp32-classic.json`で設定されているデフォルト値は21） | PCA9685のSDAピン | - |
| 乾電池バッテリーボックスの+極 | PCA9685のV+ | ブレッドボードの+極の右側を経由して配線 | 
| 乾電池バッテリーボックスの-極 | ESP32のGND  | ブレッドボードの-極を経由して配線 | 
| ESP32の5V電源 | PCA9685のVCC | ブレッドボードの+極の左側を経由して配線 | 
| ESP32のGND | PCA9687のGNDピン | ブレッドボードの-極を経由して配線 | 

<br>

ESP32-DevKitC（`config.esp32-classic.json`）の場合の、具体的な配線は下図になります。
