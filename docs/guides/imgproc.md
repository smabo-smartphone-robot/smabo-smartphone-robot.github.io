# 画像処理とは

smaboでは、スマホアプリがカメラ映像を`/camera/image/compressed`としてsmabo-brainへ送信します。

smabo-brainはこの映像を受け取り、
- 顔・物体の検出
- 検出結果をもとにした視線追従（`/look_at`）

などの高度な処理を行う役割を担います。
